# Мультиметр - Тестовое задание Unity

## Описание
Интерактивная 3D-модель мультиметра с возможностью переключения режимов и расчетом значений по закону Ома.

## Технологии
- Unity 2022.1.23f1
- C#
- TextMeshPro

## Управление
- Наведите курсор на регулятор (ручку) мультиметра
- Прокручивайте колесико мыши для переключения режимов
- Режимы: OFF → V → V~ → A → Ω

## Формулы расчета
- **Сопротивление (R)**: 1000 Ω (фиксированное)
- **Мощность (P)**: 400 W (фиксированная)
- **Сила тока (I)**: √(P / R) = 0.63 A
- **Напряжение DC (V)**: √(P × R) = 632.46 V
- **Напряжение AC (V~)**: 0.01 V (фиксированное)
